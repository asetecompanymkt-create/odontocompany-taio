# OdontoCompany Taió — Landing Prótese Protocolo

Landing page de conversão (WhatsApp) da unidade de **Taió (SC)**.
Página independente, exclusiva desta cidade. Site estático, sem build:
`index.html` + `css/` + `js/` + `assets/`.

## Editar rápido
- WhatsApp da unidade: `js/main.js` → objeto `UNITS`, chave `taio`
- Cidade fixa da página: `window.CITY = 'taio'` no `index.html`
- Cores e fontes: `css/styles.css` (`:root`)
- Vídeo e imagens: `assets/`

## Hospedagem
Deploy via Lovable conectado a este repositório GitHub.
